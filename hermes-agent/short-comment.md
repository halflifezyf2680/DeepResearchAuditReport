# Hermes Agent

Hermes Agent 说自己是"自我进化的 AI"。

背后团队融了 6500 万美元，项目上线 50 天拿了 81,000 个 Star。

但我们读完它的全部源码，结论是：**它不会自我进化，它也不会学习。**

---

## 它到底在干什么

所谓的"进化"，实际上就是：

1. **AI 给自己出考试题，自己做题，自己判卷，自己改答案** — 你觉得这套流程的分数可信吗？
2. **把所有笔记一股脑塞进脑子里** — 笔记越多，脑子越慢，token 消耗暴涨，用户有人 2 小时烧了 400 万 token
3. **AI 自己改自己的笔记，不用你同意** — 改错了也不会发现，因为没有检查机制
4. **最致命的一条：整个优化流程可能根本没生效** — 流程跑完了，输出的文件和输入一模一样，看起来在忙，实际什么都没变

---

## 一个例子帮你理解

想象一个学生说自己会"自主学习"。

实际上他的学习方法是：自己出题、自己做、自己打分，而且打分方式是看你答案里有没有出现关键词——出现就给分，不管对不对。

然后他"学完"之后有没有变好，没人知道——没有考试验证，没有打回重来的机制，变坏了也只能认。

这就是 Hermes Agent 的"自我进化"。

## 背景

- 团队：Nous Research，纽约，约 30 人
- 融资：6,500 万美元（Paradigm 领投）

- 进化子仓库：只发了 1 个版本，5 个计划阶段只完成了 1 个，社区提交的 4 个修复全部没人理
